dromblerfx:create-standalone-jar

Full name:

org.drombler.fx:drombler-fx-maven-plugin:0.15-SNAPSHOT:create-standalone-jar

Description:

Creates the executable main application JAR in the bin directory.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
<brandingId> String - The branding id.
User property is: dromblerfx.brandingId.
<srcJarFile> File - (no description)
Default value is: ${project.build.directory}/${project.build.finalName}.jar.
<targetDirectory> File - The target directory.
Default value is: ${project.build.directory}/deployment/standalone.
User property is: dromblerfx.targetDirectory.
