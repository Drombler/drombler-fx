dromblerfx:create-application-zip

Full name:

org.drombler.fx:drombler-fx-maven-plugin:0.14:create-application-zip

Description:

Generates the JAP file.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
<applicationZipFile> File - The application ZIP file.
Default value is: ${project.build.directory}/application.zip.
User property is: dromblerfx.applicationZipFile.
<targetDirectory> File - The target directory.
Default value is: ${project.build.directory}/deployment/standalone.
User property is: dromblerfx.targetDirectory.

Parameter Details

<applicationZipFile>

The application ZIP file.
  • Type: java.io.File
  • Required: Yes
  • User Property: dromblerfx.applicationZipFile
  • Default: ${project.build.directory}/application.zip

<targetDirectory>

The target directory.
  • Type: java.io.File
  • Required: Yes
  • User Property: dromblerfx.targetDirectory
  • Default: ${project.build.directory}/deployment/standalone
