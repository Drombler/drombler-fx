dromblerfx:ensure-standalone-config

Full name:

org.drombler.fx:drombler-fx-maven-plugin:1.0:ensure-standalone-config

Description:

Ensures the application config files exist in the conf directory and contain the necessary info.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
<brandingId> String - The branding id.
User property is: dromblerfx.brandingId.
<targetDirectory> File - The target directory.
Default value is: ${project.build.directory}/deployment/standalone.
User property is: dromblerfx.targetDirectory.
<userdir> String - The default user directory. The user can change the value.
Default value is: ${dollar}{user.home}/.${brandingId}/${project.version}.
User property is: dromblerfx.userdir.

Parameter Details

The branding id.
  • Type: java.lang.String
  • Required: Yes
